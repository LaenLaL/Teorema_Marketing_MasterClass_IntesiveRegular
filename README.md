# Calendario Marketing Master Class Intensivo

Pagina estatica para visualizar la estrategia de marketing de la Master Class 2 y el curso Intensivo Teorema.

## Archivos

- `index.html`: calendario interactivo y linea de tiempo.
- `estrategia_marketing_masterclass_intensivo.md`: documento maestro de estrategia.
- `preview.png`: vista previa del calendario.

## Publicacion sugerida

Subir este repositorio a GitHub y activar GitHub Pages desde la rama principal, usando la raiz del repositorio como fuente.

La pagina quedara disponible como:

```text
https://<usuario>.github.io/<repositorio>/
```
